↩ Shiny Living Dex Tracker

Este proyecto consiste en una tabla que permite almacenar datos sobre una colección de Pokémon variocolores. El botón Guardar datos descarga un archivo JSON con toda la información de la tabla. El botón Cargar datos introduce en la tabla los datos almacenados en los archivos JSON generados por el botón Guardar datos.
